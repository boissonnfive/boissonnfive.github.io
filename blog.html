Je viens de créer un blog sur GitHub grâce à un "module" que j'ai découvert dans l'"Explore" de GitHub : [Simple](http://isnowfy.github.io/simple). Comme son nom l'indique, il est très simple à utiliser. Le blog est semble-t-il hébergé par GitHub, donc gratuitement. Cool. ## Procédure With Simple and github pages you can easily create your static blog. Here is what you need to do. 1. Create a github account. 2. Create a repo named *your_user_name.github.io* (remember to check *Initilize and create README*). 3. Sign in [Simple](http://isnowfy.github.io/simple) with your github account. 4. Click *Initilize* to set up basic files for your static blog site.(maybe a little slow) 5. Click *Go* and start writing. 6. Click *New post* to create a new post and when finish writing click save to generate the static page. 7. Now you can browse the *your_user_name.github.io* site and enjoy it! ## Remarques Avoir écrit la procédure, m'a permis de me familiariser avec le Mardown de ce blog.
