Si on utilise la souris, pas de problèmes. Mais, si on utilise le clavier pour faire un copier coller ou ouvrir un fichier, le mac ne fonctionne pas pareil qu'un PC. Voici une liste des commandes fréquemment utilisées au clavier:

Sur du texte, on peut utiliser les commandes :

- copier:       <Cmd+C>
- couper:       <Cmd+X>
- coller:       <Cmd+V>
- annuler:  <Cmd+Z>
- fin:      <Ctrl+Droite>           (Il n'y a pas de touche dédiée !)
- début:        <Ctrl+Gauche>       (Il n'y a pas de touche dédiée !)
- page haut:    <Ctrl+Haut>
- page bas: <Ctrl+Bas>
- effacer:      <Fn+Backspace>      (Il n'y a pas de touche dédiée !)

Sur un fichier (l'explorateur sous mac, c'est le Finder), on peut utiliser les commandes :

- copier:       <Cmd+C>
- couper:       N'existe pas !!! Il faut déplacer les éléments d'une fenêtre à l'autre à la souris.
- coller:       <Cmd+V>
- annuler:  <Cmd+Z>         (Permet d'annuler un déplacement ou la mise à la corbeille d'un fichier)
- effacer:      <Cmd+Backspace>     (Envoie dans la corbeille)
- entrer dans un dossier:       <Cmd+↓>     (Et non pas la touche Entrée !!)
- ouvrir un fichier:            <Cmd+O>     (Et non pas la touche Entrée !!)
- remonter au dossier supérieur:    <Cmd+↑>

Pour voir les applications ouvertes et basculer d'une application à une autre : - basculer entre les applications: ou touche exposé (F3 ou F4)

Les symbôles des touches

⌃ :         Control
⌥ :         Option (Alt)
⌘ :         Command (Pomme)
⇧ :         Shift (Majuscule)
⇪ :         Caps Lock (Verrouillage majuscule)
⏎ :         Entrée (Retour charriot)
⎋ :         Esc
⏏ :         Eject
⌫ :         Backpace (Retour arrière)
⌅ :         Entrée (Retour charriot)

Quelques caractères pas évidents à trouver sur le clavier

Ç :         ⌥ ç
À :         ` ⇧ a
È :         ` ⇧ e (idem pour les u, i, etc... )
É :         ⇪ é (*)
È :         ⇪ è (*)
À :         ⇪ à (*)
Ê :         ⇪ ^ e (*)
Ë :         ⇪ ¨ e (*)
\ :         ⌥ ⇧ /
€ :         ⌥ €
{ :         ⌥ (
} :         ⌥ )
[ :         ⌥ ⇧ (
] :         ⌥ ⇧ )
| :         ⌥ ⇧ l
~ :            ⌥ n
« :         ⌥ 7
» :         ⌥ ⇧ 7
F1:         fn ⌃ F1
Fx:         fn ⌃ Fx
— :         ⌥ -
:          ⌥ &
